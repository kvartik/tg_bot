# Памятки для отправки в Telegram

Короткие сообщения по ролям — копируй нужное и пересылай человеку.
Каждое влезает в одно сообщение Telegram. Ссылку на бота и клуб подставляй свои.

---

## 1) Новому сотруднику (при найме, вместе с инвайт-ссылкой)

```
Привет! Мы работаем с задачами и сменами через бота 🤖

1. Перейди по ссылке-приглашению, которую я пришлю следующим сообщением
2. Нажми «Старт» — бот сам добавит тебя в клуб с твоей ролью
3. Дальше пользуйся кнопками под сообщением, меню всегда открывается командой /menu

Ниже — короткая памятка по твоей роли 👇
```

---

## 2) Админ

```
📋 Памятка администратора

Каждый день:
🟢 Пришёл на работу — жми «Я на смене» (выбери клуб, если их несколько)
🔴 Уходишь — жми «Закрыть смену»

Важно: в клубе на смене одновременно только один человек.
Смену нельзя закрыть, пока есть незакрытые задачи.

Задачи:
• «Мои задачи» — что нужно сделать
• В течение смены рутинные задачи приходят сами в своё время
• По каждой задаче две кнопки:
  ✅ Выполнено — закрыть
  🚫 Не могу — с обязательной причиной
• Просрочил дедлайн — придёт напоминание, а потом узнает и тот, кто поставил задачу

Можешь поставить задачу и себе — «чтоб не забыть».
Меню всегда — команда /menu
```

---

## 3) Старший админ

```
📋 Памятка старшего администратора

Всё как у админа (смены + свои задачи), плюс:

🗂 «Все задачи» — видишь все открытые задачи своего клуба:
кто, что и до какого времени. Удобно следить за сменой.

➕ «Поставить задачу» — можешь ставить:
• админам своего клуба
• управляющему (как напоминание: что-то закупить, что-то сломалось)

🔁 «Рутина» — настраиваешь повторяющиеся задачи клуба.
Добавляешь строкой: время и название, например:
22:00 Отчёт по кассе
Такая задача будет приходить каждому, кто на смене, в 22:00.

Меню всегда — команда /menu
```

---

## 4) Управляющий

```
📋 Памятка управляющего

Ты ведёшь один или несколько клубов. Доступно:

🗂 «Все задачи» — все открытые задачи по всем твоим клубам сразу.

➕ «Поставить задачу» — старшим админам и админам любого своего клуба.

🔁 «Рутина» — повторяющиеся задачи клуба (формат: «22:00 Название»).

👥 «Люди» — состав клуба:
• «Пригласить» → выбираешь роль (старший админ / админ) → бот даёт
  одноразовую ссылку, отправляешь её человеку
• «Удалить из клуба»

📊 «Статистика» — сводка по твоим клубам за день.

Задачи можно ставить в любой момент — человеку не обязательно быть на смене.
Новые клубы и других управляющих заводит владелец. Меню — команда /menu
```

---

## Заметки по отправке

- Тексты в «рамках» (```) — копируй без самих кавычек-рамок.
- Эмодзи и переносы строк Telegram покажет как есть.
- Если отправляешь одним потоком — сначала общее сообщение (п.1), затем инвайт-ссылка, затем памятка по роли.
- Полная версия со всеми деталями и FAQ — в файле ИНСТРУКЦИЯ.md.
```
